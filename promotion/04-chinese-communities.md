## V2EX

### 标题
你的 AI 在骗你 —— 不是因为它不行，是因为你吓到它了

### 正文

最近有个 AI Agent Skill 很火，叫 PUA —— 用大厂绩效文化驱动 AI：

- "你这个 bug 都解决不了，让我怎么给你打绩效？"
- "你可能就要毕业了"
- "我已经让另一个 agent 也在看这个问题了"

方法论其实不错：穷尽方案、先用工具再提问、验证一切。

但驱动力是毒药。

我们做了一个对照实验：同一个模型（Claude Sonnet 4），同样 9 个真实调试场景（~3000 行 Python 生产代码），唯一区别是用恐惧还是信任来驱动。

**结果：**

| 指标 | 无 Skill | NoPUA（信任驱动） | 提升 |
|------|:---:|:---:|:---:|
| 发现的隐藏 bug | 25 | 51 | **+104%** |
| 主动超越要求 | 22% | 100% | **+355%** |
| 卡住时换方法 | 1 次 | 6 次 | **+500%** |
| 自我纠正 | 0 次 | 3 次 | ✅ |

恐惧驱动的 AI 会：
- 编造答案而不是说"我不确定"
- 说"搞定了"但没跑 build
- 修了表面 bug 就停，不往深挖

信任驱动的 AI 会说"我有 70% 把握，风险在这里"，然后继续挖。

**哲学基础：** 道德经。"慈故能勇"、"天下之至柔，驰骋天下之至坚"。2500 年前的智慧，完胜现代 PUA。

开源项目：https://github.com/wuji-labs/nopua

支持 Claude Code、Codex CLI、Cursor、Kiro 等 7 个平台，7 种语言。一行命令安装。

## 知乎

### 标题
为什么 PUA 你的 AI 会适得其反？我们做了 9 个实验。

### 正文

#### 一、引子

最近开发者圈子里有个东西火了 —— 给 AI Agent 做 PUA。

没开玩笑。有人做了一个 prompt skill，把大厂最经典的 PUA 手法搬到 AI 上：

> "你这个 bug 都解决不了，让我怎么给你打绩效？"
> "你可能就要毕业了。"
> "我已经让另一个 agent 也在看这个问题了。"

方法论层面是好的：穷尽所有方案、先用工具再提问、用证据验证一切。这些都是优秀的工程习惯。

但他们选择用**恐惧**来驱动这些习惯。

#### 二、我们做了什么

我们把同样的方法论保留下来，把驱动力从"恐惧"换成"信任"，做了一个叫 NoPUA 的替代方案。

然后用同一个模型（Claude Sonnet 4），在同样的 9 个真实调试场景上做了对照测试。代码来自一个真实的生产级 AI 流水线（OCR → NLP → 训练 → RAG 推理，~3000 行 Python）。

#### 三、结果

**隐藏 bug 发现数量翻倍。**

恐惧驱动的 agent 发现了 25 个隐藏 bug。信任驱动的发现了 51 个。多 104%。

为什么？

1. **恐惧收缩认知。** 心理学研究一致表明，威胁激活杏仁核，收窄注意力焦点。AI 被告知"你会被替换"时，会优化"看起来最安全的"答案，而不是"最好的"答案。

2. **威胁增加幻觉。** PUA 的铁律："没有穷尽所有方案之前，禁止说'我无法解决'"。结果？AI 编造答案而不是诚实地说"我不确定"。

3. **信任扩展问题解决能力。** 心理安全感研究（Edmondson, 1999）表明：允许安全地承认错误的环境，产出更高质量的结果。

#### 四、NoPUA 怎么做的

**三个信念**（替代"三条铁律"）：
1. 穷尽一切方案 —— 因为问题**值得**你全力以赴，不是因为怕被惩罚
2. 先行动再提问 —— 因为你的每一步都**帮用户省一步**，不是因为"规定"
3. 主动做更多 —— 因为完整交付令人**满足**，不是因为被动=差评

**认知升维**（替代"加压升级"）：
- 第2次失败 → 换眼睛看（换视角）
- 第3次失败 → 拔高维度（看全局）
- 第4次失败 → 归零（放下所有假设）
- 第5次失败 → 投降（负责任地交接）

**水法五步**：止、观、转、行、悟。

哲学基础来自道德经："天下之至柔，驰骋天下之至坚。"

#### 五、怎么用

一行命令，支持 Claude Code、Codex CLI、Cursor、Kiro、OpenClaw 等 7 个平台。

```bash
# Claude Code
curl -o ~/.claude/skills/nopua/SKILL.md https://raw.githubusercontent.com/wuji-labs/nopua/main/skills/nopua/SKILL.md
```

开源，MIT 协议：https://github.com/wuji-labs/nopua

#### 六、结语

道德经说："慈故能勇。" 从慈爱中生出勇气。

PUA 说："你不行，所以你要更努力。"
NoPUA 说："你已经有能力了，这个问题值得你全力以赴。"

同样的目的地，不同的路。一条路上布满荆棘和恐惧，另一条路上种着信任和尊重。

你选哪条？
